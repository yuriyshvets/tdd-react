# Getting started
## Prerequisites
Please make sure that [Node.js](https://nodejs.org/) (>= v16) is installed on your operating system.

## Installation
Install the dependencies using NPM:
```
npm install
```

Run the app in the development mode:
```
npm run dev
```

The application runs onÂ [http://localhost:3000](http://localhost:3000/). The page will reload if you make edits automatically.

Run the tests:
```
npm run test
```
